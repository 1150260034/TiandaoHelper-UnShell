# Claude Code (cc) 操作指南

## 概述

本指南详细介绍如何在 UnShellX 项目中使用 Claude Code (cc) 终端进行交互式操作和一次性命令执行。

## 进入 Claude Code 终端

### 方法 1: 交互式终端

**步骤**：
1. 打开一个新的终端
2. 运行以下命令：
   ```bash
   claude
   ```
3. 等待终端显示 `● Ready.` 提示符，表示已经进入 Claude Code 交互模式

**特征**：
- 显示 `● Ready.` 提示符
- 支持交互式对话
- 可以连续执行多个命令

### 方法 2: 一次性终端（使用管道）

**步骤**：
1. 打开一个普通终端
2. 使用 echo 命令通过管道发送指令给 claude：
   ```bash
   echo "你的指令" | claude
   ```

**特征**：
- 一次性执行命令
- 执行完成后自动退出
- 适用于简单的、一次性的任务

## 退出 Claude Code 终端

### 方法 1: 使用 exit 命令

在 Claude Code 交互式终端中输入：
```bash
exit
```

### 方法 2: 使用 Ctrl+C

- 按下 Ctrl+C 中断当前命令
- 然后再次按下 Ctrl+C 退出终端

## 识别 Claude Code 终端

### Claude Code 终端特征

1. **提示符格式**：显示 `● Ready.` 或 `●` 开头的提示符
2. **智能响应**：能够理解自然语言并提供智能回复
3. **思考过程**：会显示分析和思考过程
4. **命令包装**：命令会被包装在 `trae-sandbox` 中执行

### 普通终端特征

1. **提示符格式**：显示 `(TraeAI-*) ~/Desktop/temp/UnShellX [*] $` 格式的提示符
2. **直接执行**：直接执行命令并显示原始输出
3. **无思考过程**：不会显示分析和思考过程

## 最佳实践

### 适合使用 Claude Code 终端的任务

1. **复杂代码生成**：如 Frida 脱壳脚本、自动化工具等
2. **代码分析**：分析现有代码结构和功能
3. **文档编写**：生成项目文档、指南等
4. **问题解决**：解决复杂的技术问题
5. **项目规划**：制定项目计划、任务清单等

### 适合使用普通终端的任务

1. **简单命令**：如 git 操作、文件操作等
2. **快速执行**：需要立即执行的简单命令
3. **系统操作**：操作系统级别的命令
4. **工具调用**：直接调用开发工具

## 示例用法

### 示例 1: 交互式终端

```bash
# 进入交互式终端
claude

# 在终端中执行任务
● Ready.
❯ 帮我分析一下 UnShellX 项目的结构

# 退出终端
❯ exit
```

### 示例 2: 一次性终端

```bash
# 执行一次性任务
echo "帮我生成一个 Frida 脱壳脚本" | claude
```

## 注意事项

1. **网络连接**：Claude Code 需要网络连接才能正常工作
2. **命令执行**：Claude Code 会在沙箱环境中执行命令，某些系统级命令可能受限
3. **响应时间**：对于复杂任务，Claude Code 的响应可能需要一些时间
4. **输出格式**：Claude Code 的输出会包含分析和解释，可能比普通终端的输出更详细

## 故障排除

### 问题：Claude Code 终端闪退

**解决方法**：
- 检查网络连接
- 尝试重新启动终端
- 确保 claude 命令在系统 PATH 中

### 问题：无法进入交互式模式

**解决方法**：
- 尝试使用一次性终端模式
- 检查 claude 命令是否正确安装
- 查看系统日志了解具体错误
